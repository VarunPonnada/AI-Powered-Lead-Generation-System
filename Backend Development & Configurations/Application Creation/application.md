# Application Creation

## Overview

A Salesforce application is a collection of tabs, objects, and tools grouped together to help users perform specific business processes. It provides a dedicated workspace where users can access related data such as Leads, Accounts, Contacts, Opportunities, Reports, and Dashboards without navigating across different parts of Salesforce. Lightning Apps can also be customized with branding, navigation items, and user access to improve productivity and user experience.

---

## Steps to Create a Lightning Application

### Step 1: Open App Manager

1. Click the **Gear (⚙️) Icon**.
2. Select **Setup**.
3. In the **Quick Find** box, search for **App Manager**.
4. Click **App Manager**.
5. Click **New Lightning App**.

---

### Step 2: App Details & Branding

Enter the application details.

- **App Name:** AI-Powered Lead Generation System
- **Developer Name:** Auto-generated (or enter AI_Powered_Lead_Generation_System)
- **Description:** Lightning application for managing AI-powered lead generation and CRM activities.

(Optional)

- Upload an application logo.
- Leave the default theme color.

Click **Next**.

---

### Step 3: App Options

Keep the default settings.

- **Navigation Style:** Standard Navigation
- **Supported Form Factors:** Desktop and Phone
- **Setup Experience:** Setup (Full Set of Setup Options)

Leave all optional checkboxes unchecked.

Click **Next**.

---

### Step 4: Navigation Items

Add the following navigation items:

- Leads
- Accounts
- Contacts
- Opportunities
- Campaigns
- Reports
- Dashboards
- Files

Arrange them in the following order:

1. Leads
2. Accounts
3. Contacts
4. Opportunities
5. Campaigns
6. Reports
7. Dashboards
8. Files

Click **Next**.

---

### Step 5: Utility Items (Desktop Only)

No utility items are required for this project.

- Leave **Utility Bar Alignment** as **Default**.
- Do not add any Utility Items.

Click **Next**.

---

### Step 6: User Profiles

Grant access to the application.

Select:

- System Administrator

(Optional)

- Custom Sales Profile
- Custom Marketing Profile
- Custom Support Profile

Move the selected profiles to **Selected Profiles**.

---

### Step 7: Save the Application

Click **Save & Finish**.

The application is now available from the **App Launcher**.

---

# Result

The AI-Powered Lead Generation System Lightning App has been successfully created.

The application provides a centralized workspace containing:

- Leads
- Accounts
- Contacts
- Opportunities
- Campaigns
- Reports
- Dashboards
- Files

Users assigned to the selected profiles can now access the application directly from the Salesforce App Launcher.
