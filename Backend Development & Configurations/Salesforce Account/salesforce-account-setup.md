# Salesforce Developer Account Setup

## AI-Powered Lead Generation System

## 1. Overview

Before developing the **AI-Powered Lead Generation System**, a Salesforce **Developer Edition Organization (Dev Org)** must be created. A Developer Org provides a free cloud environment where developers and administrators can build, customize, automate, and test Salesforce applications without affecting production data.

The Developer Organization serves as the foundation for creating custom objects, configuring fields, designing automation using Flow Builder, implementing Agentforce features, and testing the application throughout the development lifecycle.

---

## 2. Create a Salesforce Developer Account

Visit the Salesforce Developer Edition signup page:

> https://developer.salesforce.com/signup

Fill in the required registration details.

| Field | Value |
|--------|--------|
| First Name | Your First Name |
| Last Name | Your Last Name |
| Job Title | Student |
| Company | Ideal Institute of Technology |
| Country/Region | India |
| Work Email | Your Email Address |

After entering the required information, accept the Salesforce Terms and Conditions and click **Sign Me Up**.

---

## 3. Verify Your Email Address

After registration, Salesforce sends a verification email to the registered email address.

Follow these steps:

1. Open your email inbox.
2. Locate the Salesforce verification email.
3. Click **Verify Account**.
4. Continue to the account activation page.

---

## 4. Set Your Password

After email verification:

1. Create a secure password.
2. Select a security question.
3. Save the new password.
4. Log in using your registered credentials.

---

## 5. Access the Salesforce Developer Organization

After successful login, the Salesforce Developer Edition dashboard becomes available.

The Developer Organization provides access to the following Salesforce services:

- Setup
- Object Manager
- App Launcher
- Flow Builder
- Prompt Builder
- Agentforce
- User Management
- Data Import Wizard
- Reports and Dashboards

This environment will be used throughout the project to configure and develop the AI-Powered Lead Generation System.

---

## 6. Outcome

The Salesforce Developer Edition account has been successfully created and configured.

This Developer Organization will be used to:

- Create custom objects
- Configure fields and relationships
- Build Salesforce applications
- Implement validation rules
- Design automation flows
- Configure Agentforce
- Test and deploy project components

---

## Conclusion

The Salesforce Developer Edition provides a secure and fully functional development environment for building Salesforce applications. It serves as the primary workspace for implementing all modules of the **AI-Powered Lead Generation System**.
