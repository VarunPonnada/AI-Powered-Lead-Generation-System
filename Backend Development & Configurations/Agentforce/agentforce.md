# Agentforce

## Activity 1: Enable Agentforce

### Step 1: Open Einstein Setup

1. Click the **Gear Icon** → **Setup**.
2. In the **Quick Find** box, search **Einstein Setup**.
3. Click **Einstein Setup**.
4. Enable **Turn on Einstein**.
5. Refresh the page.

---

## Activity 2: Enable Agentforce

1. In **Setup**, search **Agentforce**.
2. Click **Agentforce**.
3. Enable **Agentforce**.
4. Accept the terms if prompted.
5. Wait until Agentforce is activated.

---

## Activity 3: Create a New Agent

### Step 1: Open Agent Builder

1. Go to **Setup**.
2. Search **Agentforce**.
3. Click **Agentforce Agents**.
4. Click **New Agent**.

### Step 2: Select Agent Type

Choose:

- **Service Agent** (or the agent type specified in your lab)

Click **Next**.

---

## Activity 4: Configure the Agent

Fill the following details.

**Agent Name**

```
Lead Support Agent
```

**API Name**

Auto-generated.

**Description**

```
AI agent that assists users by answering Lead-related questions and providing Lead information.
```

Click **Next**.

---

## Activity 5: Configure Agent Instructions

Enter instructions such as:

```
You are a Salesforce AI assistant.

Help users answer questions about Lead records.

Provide accurate and professional responses.

Use available Salesforce data to answer questions.

If information is unavailable, politely inform the user.
```

Click **Save**.

---

## Activity 6: Add Topics

Click **Add Topic**.

Example Topic:

```
Lead Management
```

Description:

```
Answer questions related to Lead records.
```

Save the topic.

---

## Activity 7: Add Actions

Click **Add Action**.

Add actions such as:

- Retrieve Lead Record
- Search Lead
- Update Lead
- Summarize Lead

Save the actions.

---

## Activity 8: Activate Agent

Click

```
Activate
```

Wait until the status becomes

```
Active
```

---

## Activity 9: Test the Agent

1. Open the Agent Preview panel.
2. Ask sample questions such as:

```
Show my Lead information.

Summarize this Lead.

What is the Lead Status?

Who owns this Lead?

Give me the Lead details.
```

Verify that the Agent returns correct responses.

---
