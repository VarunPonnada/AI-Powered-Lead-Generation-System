# AI-Powered Lead Generation System

## Project Overview

This project is a Salesforce-based AI-powered Lead Generation System developed using Salesforce CRM, Agentforce, Flows, Validation Rules, and Data Modeling.

The application helps businesses identify, qualify, and prioritize potential customers automatically, reducing manual effort and improving sales productivity.

## Technologies Used

- Salesforce CRM
- Agentforce
- Flow Builder
- Validation Rules
- Apex (if required)
- Data Modeling
- Reports & Dashboards
